Allmaps CLI

Usage: allmaps annotation [options] [command]

Parses and generates Georeference Annotations

Options:
  -h, --help                      display help for command

Commands:
  parse [options] [files...]      parse Georeference Annotation
  generate [options] [files...]   generate Georeference Annotation
  svg [options] [files...]        generate SVG from resource mask
  image-ids [options] [files...]  read all IIIF Image IDs from a Georeference
                                  Annotation
  help [command]                  display help for command

For more details about Allmaps, see https://allmaps.org
