Allmaps CLI

Usage: allmaps transform [options] [command]

Transforms resource coordinates to geospatial coordinates and vice versa

Options:
  -h, --help                          display help for command

Commands:
  svg [options] [files...]            transform SVG to GeoJSON
  geojson [options] [files...]        transform GeoJSON to SVG
  resource-mask [options] [files...]  transform resource masks to GeoJSON
  coordinates [options] [files...]    transform coordinates toGeo (or
                                      toResource)
  help [command]                      display help for command

For more details about Allmaps, see https://allmaps.org
